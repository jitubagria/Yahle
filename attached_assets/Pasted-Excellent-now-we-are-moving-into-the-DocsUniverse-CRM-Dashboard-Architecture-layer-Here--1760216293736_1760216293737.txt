### ⚙️ ADMIN DASHBOARD FUNCTIONALITY DETAILS

#### **A. Import & Export Management**

* Bulk import Excel for **Doctors** and **Hospitals** (CSV/XLSX parser via PHPSpreadsheet).
* Template-based mapping → Column validation before save.
* Preview table before committing.
* Option to export filtered lists (CSV or PDF) with selected columns.

#### **B. Course / Masterclass Management**

* Drag-and-drop builder for modules/lessons.
* Attach PDF, Video, HTML.
* Assign instructor (link with `doctors` table).
* Enrollment tracking (progress bar, last login).
* Generate course completion certificate (MPDF).
* Attendance & QR-based entry for masterclasses.

#### **C. Quiz & Certificate Panel**

* Add quiz, import from Excel (question, 4 options, correct key).
* Auto-randomize order.
* Result Analytics: accuracy %, fastest responders, average scores.
* Auto issue certificates via WhatsApp.
* Manual override: Admin can send certificate or re-issue.

#### **D. Jobs Management**

* Post new jobs, set expiry.
* Track applications: shortlisting, notes, change status (Accepted/Rejected).
* Export applicant resumes.
* Integration: send “Job status update” via WhatsApp or email directly from admin panel.

#### **E. AI Tools Management**

* Define tool metadata (name, description, endpoint, access role).
* Track all usage logs: user, query, timestamp, cost (if API-based).
* Approve or revoke user access.
* Add internal AI tools (like custom calculators, prompts).
* Analytics: usage frequency by tool.

#### **F. Research & Service Management**

* View all requests with status tags: Pending / In Progress / Done / Paid / Unpaid.
* Assign to provider manually or auto-assign by region/service type.
* Upload output files (editable text + PDF).
* Generate Service Completion Certificate / Invoice (MPDF).
* Audit Trail (created_by, assigned_to, completion_time).

#### **G. User Management / CRM**

* Filter users: role, city, hospital, specialization, enrolled courses, etc.
* Bulk actions: send message, assign tag, set premium, revoke AI tool.
* View full user profile including activity (courses joined, quizzes, jobs applied, etc.)
* Download reports per user (profile + stats).

#### **H. Manual Messaging Panel**

* Rich Message Composer:

  * Filter: Role + Specialty + Enrollment + City + Status.
  * Compose: Text / Image / Button.
  * Select channel: WhatsApp (via BigTos), Email, or In-App Notification.
  * Schedule or send instantly.
  * Track delivery (via BigTos response log).
* Templates: save common messages (e.g., “New Masterclass Launch”).
* Logs: who sent, when, recipients count, delivery success.

#### **I. Analytics Dashboard (Landing Page)**

Interactive charts:

* Total Doctors (by specialty, by city)
* Active Courses (with enrollment counts)
* Quiz participation trend
* Jobs & applications timeline
* Service requests count per type
* Revenue by category
* Top 10 active users
  *(Use Chart.js in admin view)*
